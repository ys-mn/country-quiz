## Overview

![screenshot](https://user-images.githubusercontent.com/16707738/92399059-5716eb00-f132-11ea-8b14-bcacdc8ec97b.png)

Demo includes a small country quiz to test your knowledge of flags and capitals of countries :)
List of countries and their information is obtained from [Rest Countries](https://restcountries.eu/).

## Features

<!-- List the features of your application or follow the template. Don't share the figma file here :) -->

This application/site was created as a submission to a [DevChallenges](https://devchallenges.io/challenges) challenge. The [challenge](https://devchallenges.io/challenges/Bu3G2irnaXmfwQ8sZkw8) was to build an application to complete the given user stories:

- I can see select an answer
- I can see if my answer is correct or incorrect
- When I answer correctly, I can move on to the next question
- When I answer incorrectly, I can see my results and try again
- I can try again
